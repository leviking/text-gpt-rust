# Install
1. add .env with:
	- OPENAI_API_KEY
	- TWILIO_ACCOUNT_SID
	- TWILIO_AUTH_TOKEN
	- INITIAL_PROMPT

2. build and run
3. go to twilio and point your number at where ever is hosting it

## Initial Prompt
this is prepended to each request to chatgpt, which lets you somewhat configure it
